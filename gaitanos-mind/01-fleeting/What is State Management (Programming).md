Components often need to change depending on the user's interaction. For
example, when a user clicks the next button in a carousel, the user interface
should show a different image.

The components need to remember things such as current/initial state

State management is the act of managing how components change with user
interaction. This is often done using a [[What is a State Variable (Programming)|state variable and setter function]].
