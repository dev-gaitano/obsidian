A Simple Linear Regression model, is a [[Linear Regression]] model that only has
one [[Independent Variable|Independent variable]] and one [[Dependent Variable|Dependent variable]] and finds the linear
function that is the most accurate at predicting the Dependent variable from the
Independent variable.
