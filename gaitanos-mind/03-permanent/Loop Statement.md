In [[Programming|programming]], a loop statement is a [[Control Flow Statement|control flow statement]], that allows a
program or a section of a program to be executed repeatedly if a boolean condition is met.

There are different kinds of loops statements;
    - [[While Loop|While loops]]
    - [[Do While Loop|Do while loops]]
    - [[For Loop|For loops]]
